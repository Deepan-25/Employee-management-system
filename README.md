Employee Management System

#How to download and run :

i) Download the Zip file and Extract it :
  it contains both front end and backend file/folder
  
**To run back end :**
Open the first Folder in the eclipse (employee-management-system folder)
Open the src/main/java and there you could see the **apllication package {[** com.emp.employee_management_system under this you will see the application file **]}** 
Right click the Application file and  click Run as there you could see the option as java Application and click it and java application starts running.

**To run the Front End :**
 Open the folder in Vs code (employee-management-ui folder)
 In the terminal type npm run dev.
 there you could see the the port link after running.
 copy paste in the browser to see the output.
